# RAG 项目

本项目是一个使用 Cursor IDE 生成的 RAG（检索增强生成）系统实现。项目旨在展示如何利用 Cursor IDE 的强大功能，快速构建和开发一个现代化的 RAG 应用。

## 项目特点
- 完全由 Cursor IDE 辅助生成和开发
- 采用现代化的技术栈和架构设计
- 实现了高效的文档检索和生成功能
- 遵循最佳实践和设计模式

## 技术栈
- 开发工具：Cursor IDE
- 版本控制：Git
- 其他技术栈将在开发过程中不断补充
