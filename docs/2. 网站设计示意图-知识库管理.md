## 文档解析配置页面
```
+----------------------------------------------------------------------+
|                                                                      |
| 数据RAG系统                                     用户名 v | 退出登录    |
|                                                                      |
+----------------------------------------------------------------------+
|        |                                                             |
| 导航栏  |  文档解析配置                                   [返回详情]   |
|        |                                                             |
| 聊天    |  选择待解析文档                                              |
|        |  +-----------------------------------------------------------+
| 智能体  |  | [✓] document1.pdf                                        |
|        |  | [✓] document2.docx                                       |
| 知识库  |  | [ ] document3.txt (已解析)                               |
|        |  +-----------------------------------------------------------+
| 用户    |                                                             |
|        |  分段方法                                                    |
|        |  +-----------------------------------------------------------+
|        |  | 分段方法选择:                                             |
|        |  | (○) 按段落分割                                           |
|        |  | (○) 按固定长度分割                                        |
|        |  | (●) 按语义分割                                           |
|        |  |                                                          |
|        |  | 最大分段长度: [  4000  ] 字符                             |
|        |  | 分段重叠长度: [  200   ] 字符                             |
|        |  | 分段标识符:   [  \n\n  ]                                 |
|        |  +-----------------------------------------------------------+
|        |                                                             |
|        |  预处理规则                              [+ 添加规则]         |
|        |  +-----------------------------------------------------------+
|        |  | 规则名称      | 正则表达式                 | 操作          |
|        |  |--------------+---------------------------+---------------|
|        |  | 删除页眉     | ^Page \d+$                | [编辑][删除]   |
|        |  | 删除特殊字符  | [^\w\s\.\,\?\!]          | [编辑][删除]   |
|        |  +-----------------------------------------------------------+
|        |                                                             |
|        |  [预览效果]                                      [开始解析]  |
|        |                                                             |
+----------------------------------------------------------------------+
```
